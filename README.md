---

## 🛠 Installation Guide
1. **Download** this repository
2. Navigate to:
   ```
   chrome://extensions  [Chrome]
   about:addons        [Firefox]
   ```
3. Enable Developer mode (toggle switch)
4. Click **Load unpacked** and select the extension folder

---

## ⚠️ Important Notes
